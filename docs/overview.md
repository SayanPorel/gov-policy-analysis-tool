# Overview

## Purpose
The Government Policy Analysis Tool democratizes access to complex legislation by using AI to parse, summarize, and analyze policies. It targets citizens, researchers, and policymakers.

## Key Features
- **Ingestion**: Pulls data from federal/state sources.
- **Parsing**: GPT-4 extracts structured data from legalese.
- **ScaleDown**: Compresses documents by 75% for efficiency.
- **Analysis**: Impact models, stakeholder identification, and correlations.
- **Engagement**: Tracks civic interactions.

## Benefits
- Makes 10,000+ page documents accessible.
- Enables cross-policy insights.
- Promotes transparency and participation.

## Technologies
- Backend: Flask, SQLAlchemy, OpenAI GPT-4.
- Frontend: React, Axios.
- AI: Hugging Face Transformers.
- Database: PostgreSQL.
