# User Guide

## Getting Started
- Launch the app (backend on 5000, frontend on 3000).
- Ingest policies via API or UI.

## Using Features
- **Search**: Enter queries in the portal.
- **Analyze**: View impact scores and charts.
- **Translate**: Paste text for simplification.
- **Metrics**: Check engagement dashboards.

## Examples
- Ingest a bill: POST to `/api/ingest` with URL.
- Search: GET `/api/search?q=climate`.
